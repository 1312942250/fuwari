# **关于SpringActuator网络风险**

### **一、风险**
项目中如果使用到SpringActuator，查看配置文件是否暴露外网访问路径

### **二、防范**
从nginx端限制外网访问
```sh
# 限制外网访问内网 actuator 相关路径
location ~ ^(/[^/]*)?/actuator(/.*)?$ {
    return 403;
}
```
### **三、验证**
输入以下连接验证外网是否能直接下载actuator的heapdump文件
> http://ip:port/actuator/heapdump
